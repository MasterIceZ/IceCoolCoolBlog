งานของคุณ :
หา string ที่มีค่า ascii มากที่สุด

ข้อมูลนำเข้า :
บรรทัดแรก n (1<=n<=1000)
อีก n บรรทัดค่อมา str[i] 

ข้อมูลส่งออก :
บรรทัดเดียว string ที่มีค่า ascii มากที่สุด

Test Case :
Input :
3
cat
dog
car

Output :
dog

หมายเหตุ : สามารถใช้โค้ดของ Cat and Dog(small)ได้แต่จะมีเงื่อนไขเพิ่มเติม ตามโจทย์
